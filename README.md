# Personal-Alarm-Clock
## Description
This project is an alarm clock that will be able to set a specific time that you want to wake up at and make a noise to wake you up or notify you that it has became that time. You will also be able to use this clock as a timer where you can set an amount of time that will then count down untill finished. It will also be able to use the ESP32 to desplay website statistics in the future. This may include things like your hackatime or Youtube subscriber count.
